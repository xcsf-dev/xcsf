◆ clset_pset_roulette()

static void clset_pset_roulette ( const struct XCSF *  xcsf,
struct Clist **  del,
struct Clist **  delprev 
)
static

Selects a classifier from the population for deletion via roulette.

If compaction is enabled and the average system error is below E0, two classifiers are selected using roulette wheel selection with the deletion vote and the rule with the largest condition + prediction size is chosen. For fixed-length representations, the effect is the same as one roulete spin.

Parameters
[in]xcsfThe XCSF data structure.
[out]delA pointer to the rule to be deleted.
[out]delprevA pointer to the rule previous to the one being deleted.
